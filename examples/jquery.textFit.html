jQuery.textFit Examples

jQuery.textFit respects existing CSS and fits within the box it is placed, regardless of padding, line-height, font-face, or letter-spacing.

These examples are run in slow motion using setInterval to show how they work. Actual sizings are invisible and average <3ms.
